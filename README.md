# Airline-Booking-System
My PHP and MySQL-based web application for managing flight bookings

## How to Run
1. Put `index.php` in the project folder in `C:\xampp\htdocs\`.
2. Import `airline_db.sql` into phpMyAdmin.
3. Start Apache and MySQL in XAMPP.
4. Open `localhost/airline_project/index.php` in your browser.

## Testing App
1. Type Name in The Box That says Passenger name
2. Choose a flight to where you want to go
3. Then click purchase ticket
4. It will then show you your tickect#, name, and destination

## Features
- **SQL Joins:** Used for retrieving passenger and flight data.
- **Transactions:** Ensures data integrity during ticket purchase.
- **Security:** Prepared statements to prevent SQL injection.
